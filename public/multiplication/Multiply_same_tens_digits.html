Multiplying two numbers with the same tens digits.

Home

If ab and ac are two numbers with a being the tens digits and b, c being ones digits then the following is true. The value of ab is 10 × a + b and the value of ac is 10 × a + c. When we multiply these together we get
(10 × a + b) × (10 × a + c) = 100 × a2 + 10 ×ac + 10 × ab + bc = 100 × a2 + 10 ×a × (b + c) + bc

The Trick:

We identify a, b and c as above and do the following.
  1. Multiply the one's digits together. Write down the one's digit of the answer and carry the tens.
  2. Add b and c. Multiply this with a. Add this to your carry and write down the ones digit of the answer. Carry the rest.
  3. Compute a2 + the carry. Write this down as the final digits.

Examples:

Find 22 × 27

  1. Here a = 2, b = 2, and c = 7. 2 × 7 = 14. Write down "4", carry "1".
  2. b + c = 9. a × 9 = 18. 18 + 1 = 19. Write "9", carry "1".
  3. a2 = 4. 4 + 1 = 5. Write "5". The answer is 594.

Find 307 × 309

  1. Here a = 30, b = 7, and c = 9. b × c = 63 so write down "3", carry "6".
  2. b + c = 16. 30 × 16 = 480. 480 + 6 = 486. Write "6" and carry "48".
  3. a2 = 900. 900 + 48 = 948. Write 948. The answer is 94,863.
